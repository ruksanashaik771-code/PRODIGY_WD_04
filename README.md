Personal portfolio
Task-04: A high-performance, Single Page Application (SPA) designed to showcase technical mastery through modern web standards.

🌟 Key Features
Single Page Application (SPA) Logic: Custom JavaScript navigation that toggles sections without page reloads.

Dynamic About Me: Includes a professional timeline and a three-pillar skill mastery grid.

Quad-Project Showcase: Four distinct project slots with interactive hover-zoom effects and tech tagging.

Visual Gallery: A dedicated space for workspace photography and professional certifications.

Interactive Contact UI: A validated contact form with a personalized "Thank You" confirmation state.

Glassmorphism Aesthetic: Modern dark-mode UI utilizing CSS variables, backdrop blurs, and Plus Jakarta Sans typography.

🛠️ Tech Stack
Core: HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+)

Design: CSS Custom Properties (Variables), Keyframe Animations

Typography: Plus Jakarta Sans (via Google Fonts)

Architecture: State-based DOM manipulation

📂 Project Structure
Plaintext
├── index.html       # Main structure and SPA sections
├── style.css        # Global styles, animations, and responsive design
